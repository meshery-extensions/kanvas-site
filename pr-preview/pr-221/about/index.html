About Kanvas

Kanvas is the visual interface for managing and collaborating on cloud native infrastructure. It is part of the Meshery ecosystem.

Kanvas empowers engineers to design, deploy, and manage cloud native infrastructure collaboratively. As a Google Workspace-like experience for Kubernetes, Kanvas transforms infrastructure management from YAML to visual intuition.
